# README

## Getting Started

1. Clone this repository

   ```bash
    git clone https://github.com/the-harry-higgins/estateably-coding-challenge.git
   ```

2. Install dependencies and get it running

      ```bash
      npm run install-run
      ```

3. Use this script for subsequent times
     ```bash
      npm run dev
     ```
